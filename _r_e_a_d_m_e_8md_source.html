25 ## Hardware

66 ## Terms:
67 ```
68 Segment: One LED in a 7-segment display
69 Digit: One 7-segment display (Value 0..9 and A..F)
70 Dot: The dot LED in a 7-segment digit
71 Pos: Print position 0...7 (MSB bit 7: left .. LB bit 0: right)
72 Radius: DEC for decimal, HEX for hexadecimal, BIN for binary
73 MaxDigits: Reserve a number of digits to print a value
74 Pad: Display fixed number of digits with 0 padding
75 Overflow: Value does not fit on the display, display minus chars
76 LSB: Most right digit, dual color LED8 or switch (SW8)
77 MSB: Most left digit, dual color LED1 or switch (SW1)
78 ```

102 ### Read 8 buttons
103 Buttons are 8-bit with bit 7 most left switch, bit 0 most right switch.
104 
105 Note: The text on the board counts from S1 to S8!
106 
107 ```c++
108 uint8_t buttons = lkm1638.getButtons();
109 ```
110 
111 ### Control 8 dual color LED's
112 Dual color LED 7 = most left (Text LED8)
113 Dual color LED 0 = most right (Text LED0)
114 
115 ```c++
116 // Turn LED 0 red on (firt LED on the right)
117 lkm1638.setColorLED(0, LedRed);
118 
119 // Turn LED 0 green on
120 lkm1638.setColorLED(0, LedGreen);
121 
122 // Turn LED 0 off
123 lkm1638.setColorLED(0, LedOff);
124 
125 // Turn multiple LEDs on, color red
126 lkm1638.colorLEDsOn(0xA9, LedRed);
127 
128 // Turn multiple LEDs off
129 lkm1638.colorLEDsOff(0x1F);
130 ```

137 ### Set/get print display position
138 The print position can be set from 0..7.
139 7 = most left digit
140 0 = most right digit
141 
142 ```c++
143 // Set postion 4
144 lkm1638.setPrintPos(4);
145 
146 // Get print position
147 uint8_t pos = lkm1638.getPrintPos();
148 ```

150 ### Print variable on 7-segment display
151 Printing starts from digit right to left with an optional maximum number of
152 digits.
153 
154 Minus '-' chars will be displayed when the value is out of range, or
155 does not fit on the display.
156 
157 Optional padding can be used to display zero's. This is for example useful to
158 print hours and minutes with fixed 2 digits.
159 
160 ```c++
161 // Print int16_t on print position
162 lkm1638.print(1234);
163 
164 // Print signed 32-bit value
165 lkm1638.print(-1234567);
166 
167 // Print 16-bit unsigned casted value
168 lkm1638.print((uint16_t)65535);
169 
170 // Print 16-bit hexadecimal unsigned value
171 uint16_t value = 0xBEEF;
172 lkm1638.print(value, HEX);
173 
174 // Print value with maximum 2 digits
175 uint8_t value = 99;
176 lkm1638.print(value++, DEC, 2);
177 
178 // Print -- when value is greater than 2 digits
179 lkm1638.print(value, DEC, 2);
180 
181 // Print 16-bit unsigned value with max 4 digits and 4 digits padding: 0009
182 uint16_t value = 9;
183 lkm1638.print(value, DEC, 4, 4);
184 
185 // Print 32-bit unsigned value
186 lkm1638.print(12345678UL);
187 
188 // Print binary uint8_t 0xA9 = 10101001
189 uint8_t value = 0xA9;
190 lkm1638.print(value, BIN, 8, 8);
191 ```
